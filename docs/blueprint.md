# **App Name**: SwiftCart

## Core Features:

- Product Catalog & Search: Browse products by category or use a search bar to find specific items quickly. Displays product images, names, prices, and availability.
- Shopping Cart Management: Users can add, remove, and update quantities of items in their cart before proceeding to checkout. Cart contents are saved per user.
- Secure Checkout Process: A streamlined checkout flow allowing users to input delivery address and choose a payment method to complete their order.
- Order Tracking: Users can view the real-time status of their placed orders, from confirmation to 'out for delivery'.
- Live Location Tracking: Customer can see the live location of delivery agent and delivery agent can see the live location of customer.
- Smart Shopping List Tool: A generative AI tool that suggests groceries based on a user's free-text input, like meal ideas, dietary preferences, or pantry needs.
- Delivery Agent Interface: Provides a user interface for delivery agents with real-time tracking capabilities for orders.
- Product Data Management: Basic admin interface for managing product listings, including adding new products, updating inventory, and setting prices using Firestore.
- Role-Based Authentication: Allows users to select their role (Customer, Admin, Delivery Agent) at the authentication page, granting appropriate access levels.

## Style Guidelines:

- Primary color: A vibrant, clean blue (#3797EB) evoking efficiency, trustworthiness, and freshness for key interactive elements.
- Background color: A very light, desaturated blue (#F0F7FA) providing a spacious and unobtrusive backdrop for product display.
- Accent color: A lively, analogous cyan-green (#0CD9C4) for calls to action, important notifications, and highlighting urgency.
- Headline and body font: 'Inter' (sans-serif) for its modern, highly readable, and neutral aesthetic, suitable for product names and descriptions.
- Use clear, simple, outline-style icons that communicate function quickly, aligning with the app's promise of speed.
- Utilize a grid-based layout for product listings, prioritizing large, appealing product images and clear pricing.
- Subtle and quick micro-interactions, like add-to-cart confirmation and order status updates, to convey responsiveness.
